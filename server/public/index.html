This is methods that we will be using

GET

https://node-emop-2.onrender.com/ - This is the static page you are now viewing and home page of the API

https://node-emop-2.onrender.com/products - This is the route for all the product in the API to be shown in the products page

https://node-emop-2.onrender.com/products/(prodID) - This is the route for a single product in the API to be shown when view details is clicked

https://node-emop-2.onrender.com/users - This route is for all the users that are in the MySQL table

https://node-emop-2.onrender.com/users/(userID) - This route is for getting the user in the MySQL table by its id

POST

https://node-emop-2.onrender.com/products - This is the route for adding product to the the API

https://node-emop-2.onrender.com/users - This is the route for adding users to the the API

PATCH

https://node-emop-2.onrender.com/users/(userID) - This route is for getting the user in the MySQL table by its id

https://node-emop-2.onrender.com/products/(prodID) - This route is for getting the product in from API by its id

DELETE

https://node-emop-2.onrender.com/users/(userID) - This route is for deleting the user in the MySQL table by its id

https://node-emop-2.onrender.com/products/(prodID) - This route is for deleting the product in from API by its id
